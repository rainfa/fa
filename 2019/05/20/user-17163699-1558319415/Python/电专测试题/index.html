电专测试题

电专测试题

画出流程图，并写出相应的Python代码：
!!!一定注意括号用英文的(), 引号为英文的"", 逗号为英文的,!!!

#01：输入一个数，输出他的后一位。
#02：输入一个数，分别输出他的前三位。
#03：输入一个不为0的数，判断它的正负性，并输出"正"或"负"。
#04：分别输入两个数，输出他们的和。
#05：分别输入两个数，判断他们的和的正负性，并输出"正"或"负"。
#06：分别输入两个数，判断他们的和的正负性，若它们的和>0：输出"正"；否则分别输出这两个数的前1位。
#07：输入一个数a，若a>0, 输出“此数为正数”，否则输出该数的后1位。
#08：输入一个数b，若b>=0, 分别输出该数的后3位，否则输出“此数为负数”。
#09：输入一个数c，若c>0, 分别输出该数的后3位，否则输出该数的后1位。
#10：分别输入两个数a,b, 判断他们的和“c”的正负性，若c>0, 输出“两数和为正”，否则分别输出该两数的后1位。
#11：分别输入两个数a,b, 判断他们的和“c”的正负性，若c>=0, 分别输出该两数的前1位，否则输出“两数和为负”。
#12：输入某同学成绩a，判断是否及格，输出“及格”（a>=60），“不及格"
#13：输入某货车总重量a，判断是否超载(a>=2000), 输出“超载”或“正常”。
#14：输入某同学成绩a，判断“优”(a>=85)，“及格”（a>=60），“不及格"，并输出。
